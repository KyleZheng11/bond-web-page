# Level 4: The Cloner

- What if we could grab the actual components, not just screenshots
- Learn by stealing from the pros
- Find websites we like, deconstruct them, and clone it
  - We learn a lot here
  - Need to understand HTML, CSS, Javascript: might be more of a skill for Kyle

How to do this

- Find a website you want to use
- Press this shortcut to get the raw html
  - Windows Chrome: ctrl+u
  - Mac Chrome: option+cmd+u
- Paste all of the html into claude code and tell claude to look at css and js files
  - example: "[pasted code] take a look at the CSS and JS files as well and use this info to better clone the original site"

How this teaches us

- Now that claude actually knows and understands actual professional level code, it can be used to teach us how design is actually done

What we need to master

- Reading and understanding source code
- identify which techniques = effects
- adapting cloned patterns to your design

The trap we dont want to fall into

- you can copy paste, but can't create. Without understanding the WHY designs work, you're limited to what already exists
